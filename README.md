Constructr
==========

ConstructrCMS based on Slim-PHP5-Framework, Twitter Bootstrap, PDO (MySQL), jQuery/Javascript and Passion!

That's it for now:

	- Frontend
	- C.R.U.D. websites
	- C.R.U.D. content elements
	- Media library
	- Media details view
	- Media trashcan
	- User management
	- User rights management
	- Generating static websites
